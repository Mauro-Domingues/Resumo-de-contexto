O que é HTML?

HyperText Markup Language é o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website

Recomendação de instalação:

Instalar o Vs Code

Estrutura básica de um documento

Cada documento HTML precisa obrigatoriamente ter:

<!DOCTYPE html/> = Na primeira linha do código para indicar o tipo do documento

<html lang="pt-bt"> <html/> = Tag principal, ela marca o início e o final do documento, pode ser atrelada ao atributo "lang" para indicar o idioma do documento

<head> <head/> = Tag usada para delimitar o cabeçalho do documento. Dentro dela outras tags de cabeçalho serão usadas, como o title, meta, link, entre outros e deve ser utilizada dentro da tag html

<body> <body/> = Tag usada para definir o corpo do documento. Dentro dela serão colocadas as tags que irão compor a nossa página e deve ser utilizada dentro da tag html

Principais aplicações dentro do head

<meta charset/> = Utilizado para definir a linguagem de codificação da página

<meta http-equiv="X-UA-Compatible" content="IE-edge"/> = Utilizada para configurar a página para ser renderizada em outra versão do Internet Explorer, o valor IE=edge vai forçar o navegador renderizar com a mais recente

<meta name="viewport" content="width=device-width, initial-scale=1.0"/> = Utilizada para fazer o site ocupar todo o espaço disponível na janela, seja em celulares, tablets ou PC's

<title> </title> = Utilizado para definir o título da página

<link rel="icon" type="image/x-icon" href="arquivo"/> = Utilizado para definir o ícone da página

<link rel="stylesheet" href="documento.css"/> = Utilizado para definir o documento externo que será utilizado para estilização da página

<style> </style> = Embora não seja uma boa prática, permite adicionar estilo à página diretamente pelo html (pode ser utilizada no head ou dentro das tags dos elementos html)

Principais aplicações dentro do body

<h> </h> = Utilizado seguido de um índice do 1 ao 6 (exemplo: h1) para inserir uma linha de título na página, o índice numérico vai determinar o tamanho da fonte

<p> </p> = Utilizado para criar um parágrafo na página

<a href="caminho"> </a> = Utilizado para criar um link de redirecionamento. Pode ser inserido o atributo target=“_blank” para fazer o link abrir em uma nova guia

Aplicações de detalhes

<b> </b> = Utilizado para criar um texto em negrito

<strong> </strong> = Utilizado para criar um texto em negrito identificável por leitores visuais

<i> </i> = Utilizado para criar um texto em itálico

<em> </em> = Utilizado para criar um texto em itálico identificável por leitores visuais

<br> = Utilizado para gerar uma quebra de linha

<hr> = Utilizada para criar uma linha horizontal no documento

<dialog> </dialog> = Utilizada para criar caixas de diálogo

<dfn>palavra</dfn> = Utilizada para identificar a palavra alvo no qual uma definição está sendo aplicada com estilização em itálico

<abbr title="definição da sigla">sigla</abbr> = Utilizada para incorporar a nomenclatura de uma sigla ao passar o cursor por cima (por exemplo a ONU)

Aplicações de incorporação de arquivos

<img src="arquivo"/> = Utilizado para inserir imagens ao documento. Pode ser adicionado o elemento alt dentro da tag de abertura para especificar uma mensagem descritiva caso a imagem não carregue

<video src="arquivo"> </video> = Utilizado para inserir vídeos ao documento. Pode ser inserido o atributo controls dentro da tag de abertura para adicionar uma barra de controle ao vídeo, alem de diversos outros recursos

<audio> </audio> = Utilizado para inserir áudios ao documento. Pode ser inserido o atributo autoplay na tag de abertura para reproduzir o áudio assim que a página for recarregada, alem de outros recursos

<iframe> </iframe> = Utilizado para incorporar outro documento à esse documento (um ótimo exemplo seria a incorporação de vídeos do youtube)

Aplicações de lista

<ul> </ul> = Utilizada para criar uma lista não ordenada

<ol> </ol> = Utilizada para criar uma lista ordenada

<li> </li> = Utilizado dentro de uma lista para criar um novo item da mesma

Aplicações de tabelas

<thead> </thead> = Utilizada para criar o cabeçalho de uma tabela

<tr> </tr> = Utilizada para criar linhas dentro de uma tabela

<th> </th> = Utilizada para criar uma célula dentro do cabeçalho de uma tabela (pode ser utilizado td porém th é mais indicada por questões semânticas)

<tbody> </tbody> = Utilizada para criar o corpo de uma tabela

<td> </td> = Utilizada para criar uma célula dentro do corpo de uma tabela

<tfoot> </tfoot> = Utilizada para criar o rodapé de uma tabela

<caption> </caption> = Utilizada para agregar uma legenda para a tabela, podendo ser utilizada no cabeçalho e no rodapé

Curiosidade: as tags th e td podem receber os atributos rowspan e colspan seguidos de um índice numérico dentro da tag de abertura para especificar que a célula preencherá mais de um espaço na tabela, sendo colspan para linhas horizontais e rowspan para linhas verticais (exemplo de estruturação: <td colspan="5">)
