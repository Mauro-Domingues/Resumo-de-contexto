O que é SQL?

Standard Query Language é uma linguagem padrão para trabalhar com bancos de dados relacionais. Ela é uma linguagem declarativa e que não necessita de profundos conhecimentos de programação para que alguém possa começar a escrever queries, as consultas e pedidos, que trazem resultados de acordo com o que você está buscando

Diferença de banco de dados relacional e não relacional

A principal diferença entre banco de dados relacional e não relacional é que o O banco de dados relacional armazena dados em tabelas, enquanto o banco de dados não relacional armazena dados no formato de valor-chave, em documentos ou por algum outro método sem usar tabelas como um banco de dados relacional

Recomendação de instalação:

Instalar o MySQL Installer 8.0.29

MySQL é um sistema gerenciador de banco de dados relacional de código aberto usado na maioria das aplicações gratuitas para gerir suas bases de dados

Instalar o MongoDB

MongoDB é um banco de dados orientado a documentos (não relacional) que possui código aberto e foi projetado para armazenar uma grande escala de dados
